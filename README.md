# Raqueta
El juego contiene una pelota y dos jugadores, si un jugador logra meter la pelota en el lado opuesto, su puntaje sube.

<img src="assets/ImgPrincipal.png"/>

## Instrucciones

### Jugador1
1. W: Sube la raqueta
2. S: Baja la raqueta

### Jugador2
1. I: Sube la raqueta
2. K: Baja la raqueta
